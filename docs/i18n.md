# Box i18n — Phase 4

Built-in internationalization (i18n) and RTL support for Box.

## Installation

```bash
bun add box-i18n
```

## Quick start

```ts
import { Box } from "box-core";
import { localeDetect, t, rtlMeta, bilingualError } from "box-i18n";

const dict = {
  en: { hello: "Hello", not_found: "Not Found" },
  fa: { hello: "سلام", not_found: "پیدا نشد" },
};

const app = new Box();

// Detect locale from Accept-Language header
app.use(localeDetect({
  default: "en",
  supported: ["en", "fa"],
}));

// Translation
app.get("/greet", (c) => c.text(t(c, "hello", dict)));

// RTL metadata
app.get("/data", (c) => c.json({
  ...rtlMeta(c.locale),
  items: [{ id: 1, name: "Item" }],
}));

// Bilingual error
app.get("/item/:id", (c) => {
  return bilingualError(c, "not_found", dict, 404);
});

app.listen({ port: 3000 });
```

## API Reference

### `localeDetect(config)`

Middleware that detects the request locale and sets `c.locale`.

**Config:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `default` | `string` | required | Fallback locale |
| `supported` | `string[]` | required | List of supported locale codes |
| `strategies` | `string[]` | `["header"]` | Detection strategies in priority order |
| `cookieName` | `string` | `"locale"` | Cookie name for `"cookie"` strategy |
| `queryParam` | `string` | `"lang"` | Query param name for `"query"` strategy |

**Strategies:** `"header"` | `"cookie"` | `"query"` | `"ip"`

### `t(c, key, dict)`

Translate a message key to the current locale. Falls back to the first dictionary locale if current locale is missing.

### `bilingualError(c, key, dict, status?)`

Return a JSON error response with the message translated. Default status: 400.

### `rtlMeta(locale)`

Returns `{ dir: "rtl" | "ltr", locale: string }` for serialized responses.

### `isRtl(locale)`

Returns `true` if the locale uses RTL direction (Persian, Arabic, Hebrew, etc.).

### `RTL_LANGUAGES`

A `Set` of RTL language codes. Extendable via `addRtlLanguage(code)`.

```ts
import { addRtlLanguage } from "box-i18n";
addRtlLanguage("syc"); // Syriac
```
