# i18n — بومی‌سازی

بسته `boxfw-i18n` تشخیص زبان، دیکشنری ترجمه، متادیتای RTL و خطاهای دو زبانه را فراهم می‌کند.

## نصب

```bash
bun add boxfw-i18n
```

## شروع سریع

```ts
import { localeDetect, t, rtlMeta, bilingualError } from "boxfw-i18n";

const dict = {
  en: { hello: "Hello", not_found: "Not Found" },
  fa: { hello: "سلام", not_found: "پیدا نشد" },
};

app.use(localeDetect({ default: "en", supported: ["en", "fa"] }));

app.get("/greet", (c) => c.text(t(c, "hello", dict)));

app.get("/data", (c) => c.json({
  ...rtlMeta(c.locale),
  items: [{ id: 1, name: "مورد" }],
}));

app.get("/item/:id", (c) => {
  return bilingualError(c, "not_found", dict, 404);
});
```

## استراتژی‌های تشخیص زبان

| استراتژی | منبع | توضیحات |
|----------|------|---------|
| `"header"` | هدر `Accept-Language` | پیش‌فرض |
| `"cookie"` | کوکی نام‌گذاری شده | `cookieName: "locale"` |
| `"query"` | پارامتر query string | `queryParam: "lang"` |
| `"ip"` | موقعیت جغرافیایی IP | placeholder |

## پشتیبانی RTL

```ts
import { isRtl, rtlMeta, addRtlLanguage } from "boxfw-i18n";

isRtl("fa");  // true
isRtl("en");  // false
rtlMeta("fa"); // { dir: "rtl", locale: "fa" }

addRtlLanguage("syc"); // اضافه کردن زبان سفارشی
```

## زبان‌های RTL داخلی

عربی (`ar`)، فارسی (`fa`)، عبری (`he`)، اردو (`ur`)، کردی (`ku`)، پشتو (`ps`) و دیگر زبان‌ها.
