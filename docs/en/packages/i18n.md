# i18n — Internationalization

The `boxfw-i18n` package provides locale detection, translation dictionaries, RTL metadata, and bilingual error responses.

## Installation

```bash
bun add boxfw-i18n
```

## Quick Start

```ts
import { Box } from "boxfw-core";
import { localeDetect, t, rtlMeta, bilingualError } from "boxfw-i18n";

const dict = {
  en: { hello: "Hello", not_found: "Not Found" },
  fa: { hello: "سلام", not_found: "پیدا نشد" },
};

const app = new Box();

// Detect locale from Accept-Language header
app.use(localeDetect({
  default: "en",
  supported: ["en", "fa"],
}));

// Translated response
app.get("/greet", (c) => c.text(t(c, "hello", dict)));

// RTL metadata
app.get("/data", (c) => c.json({
  ...rtlMeta(c.locale),
  items: [{ id: 1, name: "Item" }],
}));

// Bilingual error
app.get("/item/:id", (c) => {
  return bilingualError(c, "not_found", dict, 404);
});
```

## Locale Detection

Detection strategies (checked in order):

| Strategy | Source | Config |
|----------|--------|--------|
| `"header"` | `Accept-Language` header | Default |
| `"cookie"` | Named cookie | `cookieName: "locale"` |
| `"query"` | URL query param | `queryParam: "lang"` |
| `"ip"` | IP geolocation | Placeholder — falls back to default |

```ts
app.use(localeDetect({
  default: "en",
  supported: ["en", "fa", "ar"],
  strategies: ["header", "cookie", "query"],
  cookieName: "locale",
  queryParam: "lang",
}));
```

## Translation

```ts
const dict = {
  en: { welcome: "Welcome, {{name}}!", goodbye: "Goodbye" },
  fa: { welcome: "{{name}}! خوش آمدید", goodbye: "خداحافظ" },
};

app.get("/welcome", (c) => {
  const message = t(c, "welcome", dict);
  return c.json({ message });
});
```

## RTL Support

```ts
import { isRtl, rtlMeta, addRtlLanguage } from "boxfw-i18n";

isRtl("fa");  // true
isRtl("en");  // false

rtlMeta("fa"); // { dir: "rtl", locale: "fa" }
rtlMeta("en"); // { dir: "ltr", locale: "en" }

// Add custom RTL languages
addRtlLanguage("syc"); // Syriac
```

## Bilingual Errors

```ts
app.get("/item/:id", (c) => {
  const item = findItem(c.params.id);
  if (!item) return bilingualError(c, "not_found", dict, 404);
  return c.json(item);
});
```

## API

| Function | Description |
|----------|-------------|
| `localeDetect(config)` | Middleware — detects locale from request |
| `t(c, key, dict)` | Translate a key to current locale |
| `bilingualError(c, key, dict, status?)` | JSON error with translated message |
| `rtlMeta(locale)` | `{ dir, locale }` for response metadata |
| `isRtl(locale)` | Check if locale is RTL |
| `addRtlLanguage(code)` | Add custom RTL language |
| `RTL_LANGUAGES` | Built-in RTL language codes |

## Built-in RTL Languages

Arabic (`ar`), Aramaic (`arc`), Divehi (`dv`), Persian (`fa`), Hausa (`ha`), Hebrew (`he`), Khowar (`khw`), Kashmiri (`ks`), Kurdish (`ku`), Pashto (`ps`), Urdu (`ur`), Yiddish (`yi`)
