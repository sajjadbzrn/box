# test-proj

A [Box Framework](https://github.com/sajjadbzrn/box) project.

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun dev
```

The server will start at `http://localhost:3000`.


## Testing

```bash
bun test
```



## Database

```bash
# Generate migration
bun db:generate

# Run migration
bun db:migrate
```
